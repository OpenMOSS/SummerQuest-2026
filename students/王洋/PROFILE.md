## CS336 学习计划

- 当前基础：有 Audio Language Model、音频模型和多模态数据处理相关项目经验；需要进一步补齐从头训练语言模型、系统性能分析、Scaling 和对齐方法。
- A1 Basics：从头实现并测试 tokenizer、Transformer 和训练循环，重点理解这些组件如何扩展到音频 token 与多模态输入。
- A2 Systems：学习使用 profiler 分析训练与推理中的计算、显存、数据吞吐和通信瓶颈，并关注 Voice Agent 的流式推理延迟。
- A3 Scaling：理解模型规模、数据规模和计算预算之间的关系，学习设计适用于音频与多模态任务的受控 Scaling 实验。
- A4 Data：系统练习数据清洗、去重、过滤、采样和质量评估，并关注音频与文本的时间和语义对齐。
- A5 Alignment：理解监督微调、偏好优化和强化学习方法，重点分析 Voice Agent 的交互自然度、稳定性和奖励投机问题。
- A6 Harness：待题目发布后确定，并继续采用自动化测试、固定评测和失败样例分析。
